Справка по заполнеию плана
  • План проекта может изменяться долько в момент когда он не запущен.
  • План проекта делится на стадии и задачи. В 1 момент времени может быть запущена только 1 стадия.
  • В каждой стадии обязательно должна быть 1 и только 1 задача с зависимостью "Первая задача": при запуске плана и стадии именно с этой задачи начнется выполнение задач.
  • Задачи и стадии из плана могут быть только завершены.
  • Можно изменять после первого запуска, но для этого следует его остановить
  • Задачи и стадии, у которых статус отличный от "план" изменениям уже не подлежат.
